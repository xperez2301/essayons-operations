EOMS v1.3.4 Settings Credentials Upgrade

Adds:
- RMS username stays saved in Settings
- RMS password can be saved once and preserved
- Remember RMS Credentials checkbox
- Show/Hide password button
- Password saved indicator
- Last successful RMS login time
- RMS connection status
- Test RMS Login button directly on Settings page

Use:
1. Open /settings
2. Enter RMS username
3. Enter RMS password
4. Keep Remember RMS Credentials checked
5. Save Settings
6. Test RMS Login
